Portfolio Website Creation

This is my first website, built from scratch using HTML, CSS, and JavaScript hosted on Github pages.

Overview

I created this to learn the basics of web development and to showcase my work.

Goals

  • Create a lightweight, fast-loading static site.
  • Implement a consistent design system using CSS custom properties.
  • Ensure mobile-first, responsive layouts with Flexbox and Grid.
  • Deploy seamlessly to GitHub Pages with automatic updates.

Development Process

  1. Plan: Sketched layout and navigation structure.
  2. Build: Wrote semantic HTML, styled with CSS custom properties, and added interactivity with vanilla JavaScript.
  3. Deploy: Pushed code to GitHub and enabled GitHub Pages for live hosting.

Tech Stack

  • HTML5 & Semantic Markup
  • CSS3 (Custom Properties, Flexbox, Grid)
  • JavaScript ES6+
  • Git & GitHub Pages

Challenges

Learning responsive design and CSS layout techniques was challenging at first, but I have grown more confident the more I worked on this website.

Outcome & Next Steps

This site is now my professional portfolio, as I continue to enhance my own skills this website will be updated with projects I have worked on.
