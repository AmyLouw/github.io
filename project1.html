Portfolio Website Creation

Designing and developing my personal portfolio using HTML, CSS, and JavaScript, then deploying via GitHub Pages.

Overview

This project involved building a professional, responsive portfolio site from scratch. It showcases my skills, projects, and background in a clean, modern design. Key goals included accessibility, performance, and maintainability.

Goals

  • Create a lightweight, fast-loading static site.
  • Implement a consistent design system using CSS custom properties.
  • Ensure mobile-first, responsive layouts with Flexbox and Grid.
  • Deploy seamlessly to GitHub Pages with automatic updates.

Development Process

  1. Planning & Wireframes: Sketched site structure, navigational flow, and key sections (Home, About, Projects, Contact).
  2. HTML Structure: Wrote semantic markup with reusable components (header, sidebar, hero, cards).
  3. Design Tokens: Defined colors, spacing, typography scales in :root for consistency.
  4. Responsive Styling: Used media queries, flex and grid utilities to adapt layout across breakpoints.
  5. Interactive Elements: Added smooth-scroll, mobile navigation toggle via vanilla JavaScript.
  6. Testing & Optimization: Verified accessibility (alt text, ARIA roles), performance (image optimization, minification), and cross-browser support.
  7. Deployment: Pushed to GitHub repo username.github.io and configured Pages settings.

Tech Stack

  • HTML5 & Semantic Markup
  • CSS3 (Custom Properties, Flexbox, Grid)
  • JavaScript ES6+
  • Git & GitHub Pages
  • Image Optimization (WebP, SVG icons)

Challenges & Learnings

CSS Variable Scope: Learned to declare design tokens in :root for global access.

Responsive Navigation: Built a fixed sidebar with a mobile-friendly toggle for narrow viewports.

Dark Mode Support: Implemented prefers-color-scheme media queries for theme adjustments.

Deployment Workflow: Gained experience with GitHub Pages, custom domains, and HTTPS setup.

Outcome & Next Steps

The portfolio site now serves as my professional gateway, effectively highlighting my work and skills. Future enhancements include adding case study deep-dives, integrating a blog with static site generation, and experimenting with advanced CSS animations.
